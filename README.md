# hqqich工具

## 引入

```xml

<project>
    <repositories>
        <repository>
            <id>
                io.github.hqqich
            </id>
            <name>
                hqqich-tool-all
            </name>
            <url>
                https://hqqich.github.io/hqqich-tool/maven-repo/
            </url>
        </repository>
    </repositories>
    <dependencies>
        <dependency>
            <groupId>
                io.github.hqqich
            </groupId>
            <artifactId>
                hqqich-tool-all
            </artifactId>
            <version>
                1.0.3
            </version>
        </dependency>
    </dependencies>
</project>
```

| 模块                       | 名称       | 版本    |
|--------------------------|----------|-------|
| hqqich-tool-all          | 引用所有工具   | 1.0.3 |
| hqqich-tool-cat          | 加解密工具    | 1.0.3 |
| hqqich-tool-csv          | csv工具    | 1.0.3 |
| hqqich-tool-id-generator | id生成器    | 1.0.3 |
| hqqich-tool-kotlin-ext   | kotlin扩展 | 1.0.3 |
| hqqich-tool-file         | 文件操作相关   | 1.0.3 |
